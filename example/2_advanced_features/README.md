# 2_advanced_features

本章节覆盖较高阶的 Runtime 能力，适合在掌握基础特性后继续学习内建任务、Kernel 执行、模型运行实例、回调、通知、TDT 以及高级执行控制。

## 样例列表

- [built_in_task](./built_in_task/README.md)：Runtime 内建任务，包括 Reduce 和随机数任务。
- [callback](./callback/README.md)：Stream 回调、HostFunc 回调和异常回调。
- [kernel](./kernel/README.md)：Kernel 二进制加载、参数组织、执行和资源限制。
- [label](./label/README.md)：Label、流跳转和设备端流程控制。
- [model_ri](./model_ri/README.md)：模型运行实例的捕获、更新与切换。
- [notify](./notify/README.md)：进程间 Notify 共享和流间同步。
- [tdt_buffer](./tdt_buffer/README.md)：TDT Buffer 的共享、引用和链式组织。
- [tdt_channel](./tdt_channel/README.md)：TDT Channel 的数据传输、容量控制和数据结构解析。
- [tdt_queue](./tdt_queue/README.md)：TDT Queue 的属性配置和路由管理。
